== Description ==
MaterialX it's a great child theme based on the structure of Metro-CreativeX designed by Themeisle but with the awesome material design applied. Now you can enjoy a great experience from material design.

== Installation ==
Please note that you need to install first the main theme, Metro-CreativeX https://wordpress.org/themes/metro-creativex/ .
